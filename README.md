# Predicting Social Media Addiction (Machine Learning Project)

This project was completed for PSTAT 131 at UC Santa Barbara.

## Objective
Build machine learning models to predict social media addiction scores based on behavioral and demographic variables.

## Dataset
705 observations including:
- Daily social media usage
- Sleep hours
- Mental health score
- Most used platform
- Academic performance impact

## Models Used
- Linear Regression
- Interaction Linear Model
- K-Nearest Neighbors
- Ridge Regression
- Lasso Regression
- Pruned Regression Tree

## Tools
- R
- tidymodels
- ggplot2

## Evaluation
Models were compared using cross-validation RMSE.
